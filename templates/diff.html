1. BASIC DIFFERENTIATION RULES


d/dx(c) = 0
d/dx(x) = 1
d/dx(x²) = 2x
d/dx(x³) = 3x²
d/dx(xⁿ) = nxⁿ⁻¹
d/dx(√x) = 1/(2√x)
d/dx(1/x) = -1/x²
d/dx(x⁻ⁿ) = -n/xⁿ⁺¹

2. ALGEBRAIC RULES


d/dx [cf(x)] = c·f'(x)
d/dx [f(x)+g(x)] = f'(x)+g'(x)
d/dx [f(x)-g(x)] = f'(x)-g'(x)

3. PRODUCT RULE


d/dx [u·v]
= u(dv/dx) + v(du/dx)

4. QUOTIENT RULE


d/dx [u/v]
= (v·du/dx - u·dv/dx)/v²

5. CHAIN RULE


d/dx[f(g(x))]
= f'(g(x))·g'(x)

6. EXPONENTIAL FUNCTIONS


d/dx(eˣ) = eˣ
d/dx(eᵘ) = eᵘ·du/dx
d/dx(aˣ) = aˣ ln(a)
d/dx(aᵘ) = aᵘ ln(a)·du/dx

7. LOGARITHMIC FUNCTIONS


d/dx(lnx) = 1/x
d/dx(logₐx) = 1/(xln(a))
d/dx(lnu) = (1/u)·du/dx
d/dx(logₐu) = (1/(uln(a)))·du/dx

8. TRIGONOMETRIC FUNCTIONS


d/dx(sinx) = cosx
d/dx(cosx) = -sinx
d/dx(tanx) = sec²x
d/dx(cotx) = -cosec²x
d/dx(secx) = secx tanx
d/dx(cosecx) = -cosecx cotx
